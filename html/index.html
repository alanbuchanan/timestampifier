Alanbuchanan's Timestampifier

This is a small service for delivering UNIX timestamps and normal dates as a JSON file.

Go ahead and type in a UNIX timestamp, or a normal date, into the search bar!

Example input:

https://timestampifier.herokuapp.com/3%20Feb%202016 https://timestampifier.herokuapp.com/1454457600

Example output

{ unixTimestamp: 1454457600, naturalLanguageDate: "3 February 2016" }
